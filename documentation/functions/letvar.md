# let / var
**In SGC++, you can use `let` or `var` to declare read/write variables that can be changed unlike `const`.**

**Examples**:

`let example = "a"`<br>
`let number = 0`<br>
`var example = "a"`<br>
`var number = 0`<br>

**You can also do stuff like**:

`let name = gReadln("what is your name?")`<br>
`name = name.lower()`
